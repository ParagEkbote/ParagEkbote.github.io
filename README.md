# Changelog of Open Source Contributions

This repository serves as a public, living contribution changelog: a curated record of merged pull requests, documentation fixes, examples, and other open-source contributions.

A complete list of my contributions is available in the page: [**contributions.md**](contributions.md).


## Purpose


This project exists to maintain a clear, searchable changelog of my open-source contributions. It helps with:

- Transparency: preserving context for why changes were made and where they landed.
- Attribution: making contributions discoverable for maintainers and collaborators.
